# RYANX BOT
BOT WHATSAPP PARA TERMUX

# COMANDOS
```bash
> pkg install git
> pkg install nodejs
> pkg install ffmpeg
> git clone https://github.com/RYANXP/ryanx-bot
> cd ryanx-bot
> npm i
> node index.js
> Escaneia o Qr e pronto :)
```
